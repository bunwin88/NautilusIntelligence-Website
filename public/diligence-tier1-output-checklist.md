# Nautilus Intelligence - Tier 1 Document Info Parser Report Output Checklist

Tier 1 is a structured parser/report package, not the full Diligence OS.

## Core outputs
1. Scope and limitations
2. Documents reviewed
3. Document inventory
4. Document type classification
5. Coverage-line summary
6. Plan/vendor map
7. Rates, premiums, and contribution structure
8. Plan design summary
9. Plan design comparison
10. Missing information checklist
11. Evidence notes and source references
12. DRL status when a request list is provided
13. Seller-response reconciliation when responses are provided
14. Call-note reconciliation when call notes are provided
15. Compliance review items based on document support
16. Risk / open issue register
17. Executive summary support
18. Formal report support and evidence-linked workpapers

## Not included in Tier 1
- Full deal operating system
- Email ingestion and automated task/reminder management
- Plan marketing package operating workflow
- Carrier/vendor outreach workflow management
- Live client-facing dashboard
- Legal, tax, ERISA, actuarial, investment, or benefits advice
