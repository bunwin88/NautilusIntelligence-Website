# NautilusIntelligence.ai — Complete Diligence Website Repository

This repository is the complete production source for **NautilusIntelligence.ai**.
It positions Nautilus Intelligence around **Nautilus Diligence**, the benefits diligence workflow for M&A transactions.

## Repository scope

Included:
- Nautilus Intelligence home page
- Nautilus Diligence platform page
- Diligence process page
- Diligence resources and downloadable templates
- Free Fit Review intake page
- Contact, legal, and 404 pages
- Required branding assets
- GitHub Pages deployment workflow
- `CNAME` for `nautilusintelligence.ai`
- Lightweight redirect pages for historical Labs URLs

Excluded:
- Quantum research and datasets
- Market and quantitative research
- Robotics and hardware research
- Agriculture, RF, forestry, and conservation work
- Labs research updates
- Membership/library backend experiments
- Supabase files and unused JavaScript

Those excluded areas belong in the separate **NautilusIntelligenceLabs.com** repository.
