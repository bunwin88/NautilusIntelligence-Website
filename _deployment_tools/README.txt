EASIEST RELIABLE UPLOAD
1. Extract the ZIP completely.
2. Open _deployment_tools.
3. Double-click VERIFY_COMPLETE_PACKAGE.bat.
4. Double-click UPLOAD_COMPLETE_REPO_TO_GITHUB.bat.
5. Type DEPLOY when prompted and complete the GitHub browser sign-in if requested.

The upload script replaces the remote main branch and includes the hidden .github folder. No file creation or code pasting is required.

After upload, keep GitHub Settings > Pages > Source set to GitHub Actions. The workflow will run automatically.
